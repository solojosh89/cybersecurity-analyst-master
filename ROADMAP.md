# Roadmap

A phased plan for building cybersecurity analyst skills. Check items off as you complete them.

## Phase 1 — Foundations
- [ ] Networking fundamentals (OSI/TCP-IP, subnetting, routing)
- [ ] Operating systems (Linux fundamentals, Windows fundamentals)
- [ ] Scripting basics (Bash, Python)
- [ ] Core security concepts (CIA triad, threat modeling, risk basics)

## Phase 2 — Core Security Skills
- [ ] Security tools (Wireshark, Nmap, Burp Suite)
- [ ] SIEM fundamentals (Splunk / ELK basics)
- [ ] Log analysis and detection engineering basics
- [ ] Vulnerability scanning and management

## Phase 3 — Specialization Tracks
- [ ] Security Operations Center (SOC) analyst path
- [ ] Incident response and digital forensics
- [ ] Penetration testing / offensive security basics
- [ ] Governance, risk, and compliance (GRC) basics

## Phase 4 — Applied Projects
- [ ] Build a home lab (see `labs/`)
- [ ] Complete 2–3 portfolio projects (see `projects/`)
- [ ] Contribute writeups and notes back to `notes/`

## Phase 5 — Certification & Career Prep
- [ ] Target certification(s) (e.g. Security+, CySA+, eJPT)
- [ ] Resume and portfolio review
- [ ] Mock interviews

## Ongoing
- [ ] Keep `notes/` updated with cheat sheets and lessons learned
- [ ] Review and refresh `curriculum/` modules periodically
