# Module 01: Networking

**Status:** IN PROGRESS

**Sequence:** Theory-first (exception to the usual investigate-first pattern - see note below)

## Why this module works differently

Module 00 (processes) worked investigate-first because "a running program" was already intuitive. Networking terms (port, TCP, established state) aren't - Taiwo is starting from zero here. So Module 01 uses a prerequisite-gated sequence: each lesson unlocks the next, and the hands-on lab stays locked until the vocabulary to actually understand it exists. This was an explicit decision on 2026-07-19, documented, not a silent policy change. Module 02 onward defaults back to investigate-first unless there's a similar reason not to.
