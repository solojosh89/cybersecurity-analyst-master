# Quiz: Module 00

Answer in your own words in notes.md, then check yourself against lesson.md.

1. A colleague says "svchost.exe is running, so this process is safe." What's wrong with that reasoning, and what would you check instead?
2. You find a process named `chrome.exe` running from `C:\Users\Public\chrome.exe` instead of `C:\Program Files\Google\Chrome\Application\`. Is the name enough to clear it? What's your next step?
3. What does a process's ParentProcessId tell an investigator that its name doesn't?
4. Why can one file on disk (`svchost.exe`) correspond to dozens of running processes at once, while most programs only ever run as one process at a time?
5. In your own words: what is the difference between a program and a process?
