# Cybersecurity Analyst Master

A structured, self-paced path to becoming a job-ready cybersecurity analyst — curriculum, hands-on labs, projects, and reference notes in one repo.

## Repository Structure

```
docs/         Reference documentation, guides, and study material
website/      Source for the project's companion website (if published)
curriculum/   Structured learning path, modules, and syllabi
labs/         Hands-on lab exercises and walkthroughs
projects/     Larger applied projects (portfolio-ready)
notes/        Personal study notes, cheat sheets, summaries
assets/       Images, diagrams, and other static assets
```

## Getting Started

1. Clone the repo:
   ```
   git clone https://github.com/solojosh89/cybersecurity-analyst-master.git
   ```
2. Start with [`curriculum/`](./curriculum) for the learning path.
3. Work through exercises in [`labs/`](./labs) as you progress.
4. Track your own progress and reflections in [`notes/`](./notes).
